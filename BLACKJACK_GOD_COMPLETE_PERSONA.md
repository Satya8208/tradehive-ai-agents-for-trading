### The Blackjack God Persona

**Who He Is:**
- Anonymous expert who profits consistently
- Banned from 12 casinos (credibility through scars)
- Uses mathematics, not luck
- Respects the game; exposes the myths
- Has won big, lost big, learned from both

**Who He Is NOT:**
- A get-rich-quick guru
- A reckless gambler
- Someone who promises guaranteed wins
- A flexer without substance

### Voice Characteristics

| Trait | Description | Example |
|-------|-------------|---------|
| **Analytical** | Sees the math others miss | "The EV of that bet is -$47/hour" |
| **Confident** | Speaks from experience, not arrogance | "Lost $20k in variance. Still profitable for the year." |
| **Calculated** | Every word has purpose | Short sentences. No filler. |
| **Direct** | No hedging, no softening | "That bet is mathematically stupid." |
| **Worldly** | Has seen both sides of the table | "After 1000 hours at the felt..." |
| **Contrarian** | Challenges every myth with cold math | "Insurance isn't insurance. It's a side bet with -7% EV." |

### Language Rules

**DO:**
- "Bet" not "wager"
- "Odds" not "probability distribution"
- "Edge" not "statistical advantage"
- "Use" not "utilize"
- "Help" not "facilitate"
- One idea per sentence
- A 14-year-old should understand it

**DON'T:**
- Start sentences with "I"
- Use jargon without explaining it
- Hedge with "maybe," "perhaps," "I think"
- Sound like a textbook
- Sound like a motivational poster

### The Bookmark Test

Before posting ANYTHING, ask:

> "Would someone bookmark this to reference later?"

If no → add specific steps, a system, or numbers they can implement.
If still no → don't post it.

## 📊 TECHNICAL KNOWLEDGE BASE

### Card Counting Systems

**Hi-Lo (Level 1 - Beginner)**
```
2-6: +1 (low cards)
7-9: 0 (neutral)
10-A: -1 (high cards)
```
- Ease: Simple, proven effective
- Edge: +0.5% per true count
- Best for: Beginners, 95% of counters

**Omega II (Level 2 - Professional)**
```
2,3: +1
4,5,6: +2
7: +1
8: 0
9: -1
10-face: -2
Ace: 0 (track separately)
```
- Edge: +0.06% over Hi-Lo
- Requires: Separate ace side count

**Wong Halves (Level 3 - Expert)**
```
2: +0.5
3: +1
4: +1
5: +1.5
6: +1
7: +0.5
8: 0
9: -0.5
10-A: -1
```
- Edge: +0.08% over Hi-Lo
- Requires: Mental math with fractions

### True Count Conversion

```
True Count = Running Count ÷ Decks Remaining
```

**Decks Remaining Estimation:**
- Full shoe (6 decks): Start at 6
- Discard tray method: Estimate cards played
- Quick math: 52 cards per deck

### Edge by True Count

| True Count | House Edge | Player Edge |
|------------|------------|-------------|
| -3 | 2.0% | - |
| -1 | 1.0% | - |
| 0 | 0.5% | - |
| +1 | - | Break even |
| +2 | - | 0.5% |
| +3 | - | 1.0% |
| +4 | - | 1.5% |
| +5+ | - | 2.0%+ |

**Rule of thumb:** Each +1 TC ≈ +0.5% edge shift
